Установка: Microsoft SQL Server 2019
1.	Требуется выход в интернет и переименовать ПК.
2.	SQL2019-SSEI-Express.exe (Далее, далее, далее…)
3.	Выбрать раздел «Новая установка изолированного экземпляра SQL Server или добавление компонентов к существующей установке»
4.	Галочку в «Я принимаю условия лицензии и…».
5.	В «Выбор компонентов» снять галочки с: «Службы машинного обучения и расширения языков», «R», «Python», «Java».
6.	В настройках экземпляра выбрать: «Именованный экземпляр» - SQLExpress, «Идентификатор экземпляра» - SQLEXPRESS.
7.	В пункте «Настройка ядра СУБД» выбрать смешанный режим и задать пароль.

1.	SQL Server 2019 Configuration Manager.exe – устанавливаем.
2.	В сетевой конфигурации включить TCP/IP и в разделе «IPAII» TCP-порт вписать 1433.
3.	В службах SQL Server её нужно перезапустить.
4.	В «Обозраветель…» поставить в разделе служба режим запуска – АВТОМАТИЧЕСКИ.
5.	В свойствах SQL Server в разделе «FILESTREAM» - везде поставить галки.
6.	Службу SQL Server нужно перезапустить.

1.	Затем в Server Management Studio заходим от администратора.
2.	Проверка подлинности SQL Server Логин: sa, пароль иксы.
3.	Создаем новый запрос и выполняем его EXEC sp_configure filestream_access_level, 2 RECONFIGURE
